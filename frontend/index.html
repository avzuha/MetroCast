How it Works
MetroCast combines live check-in/out flow from DMRC tap data with historical crowd patterns per hour and day type. An ARIMA model forecasts short-term changes, blended with a time-of-day baseline calibrated for Delhi Metro peak patterns.

Privacy Guarantee
All data used is aggregate only — no individual trips, card IDs, or personal identifiers are ever collected, stored, or transmitted. This app is fully open source.
MetroCast
v1.0.0 — Open Source
MetroCast is a free, open-source crowd intelligence platform for Delhi Metro. It helps commuters choose less crowded trains by combining real-time check-in data with ML predictions. No personal data is collected, stored, or processed. All predictions are aggregate statistics only.
Privacy by Design
No Personal Data
Zero user accounts, zero location tracking, zero history stored.
Aggregate Only
Only station-level counts used. No individual journeys tracked.
Open Source
Full source code public. Anyone can audit, contribute, or fork.
