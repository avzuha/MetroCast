About MetroCast

MetroCast is a free, open-source crowd intelligence platform for Delhi Metro. It helps commuters choose less crowded trains by combining real-time check-in data with ML predictions.

No personal data is collected, stored, or processed. All predictions are aggregate statistics only.

Open Source - MIT License Backend: Python (FastAPI) + scikit-learn / TensorFlow Lite. Frontend: Vanilla HTML/JS PWA. No login required.
